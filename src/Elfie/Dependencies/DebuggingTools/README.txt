Debugging Tools for Windows.
- Used to find and download symbols and source files from symbol servers.
- This version is 10.0.10586.0.
- Included with the Windows Driver Kit [WDK]
- Get from https://msdn.microsoft.com/en-us/windows/hardware/dn913721.aspx
- Installed to %Program Files (x86)%\Windows Kits\10

- Get the following binaries:
%Program Files (x86)%\Windows Kits\10\Debuggers\x86
symchk.exe
dbgcore.dll
dbghelp.dll
dbgeng.dll
dbgmodel.dll
srcsrv.dll
symsrv.dll
SymbolCheck.dll

%Program Files (x86)%\Windows Kits\10\Debuggers\x86\srcsrv
srctool.exe
pdbstr.exe

- Symbol Source Locations:
http://srv.symbolsource.org/pdb/Public
https://nuget.smbsrc.net
http://referencesource.microsoft.com/symbols
http://msdl.microsoft.com/download/symbols
